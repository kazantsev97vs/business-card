
    Визитная карточка

    - Пользователь (владелец)
    - карточка

    Страницы:

        1) страница регистрации пользователя
        2) страница просмотра/редактирования пользователя

        3) страница создания карточки
        4) страница просмотра/редактирования карточки

        Страницы просмотра/редактирования будут разбиты на под страницы:

        Раздел "Контактные данные":

        Страница просмотра этого раздела будет включать отображение полей

        - Имя
        - Фамилия
        - Город проживания

        А страница редактирования будет отображать редактирование этих полей

        Страницы регистрации будут использовать страницы для редактирования внутри себя

    Модели:

        1) пользователь
        2) карточка

    Установка:
        npm install react-router-dom
        npm install redux
        npm install react-redux
